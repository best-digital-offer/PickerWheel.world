## How to Deploy (Choose ONE)

### Option 1: GitHub Pages (Easiest)
1. Create new GitHub repository
2. Upload `index.html` file
3. Go to Settings > Pages
4. Select "Deploy from branch" > "main"
5. Your site is live at: `https://username.github.io/repository-name`

### Option 2: Netlify (Drag & Drop)
1. Go to netlify.com
2. Drag the `index.html` file to their deploy area
3. Get instant live URL

### Option 3: Vercel (Free)
1. Go to vercel.com
2. Import from GitHub or upload file
3. Get instant live URL

### Option 4: Any Web Host
- Upload `index.html` to any web hosting service
- Rename to `index.html` if needed
- Works on any server
